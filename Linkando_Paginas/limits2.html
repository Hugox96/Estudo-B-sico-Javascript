Entendendo os Limites no Cálculo

Nesta página, vamos explicar como funcionam os limites no cálculo.

O que é um limite?

Em cálculo, um limite representa o valor para o qual uma função se aproxima à medida que a variável independente se aproxima de um determinado valor.

Exemplo de Limite

Vamos considerar a função f ( x ) = x 2 . O limite de lim x → 2 f ( x ) representa o valor de lim x → 2 f ( x ) quando x se aproxima de 2:

lim x → 2 f ( x ) = lim x → 2 x 2 = 2 2 = 4
